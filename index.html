HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the base of the website.
  • Elements and Tags are similar, but two tags make up one element.

CSS

  • CSS is the style behind the website
  • CSS is used for color, boxes, font style and size.
  • CSS can be done internal, but is more often done in and external file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git to start in windows just type start or code.
  • git allows you to work on different local branches so you can add it to the main branch later.

JavaScript

  • JavaScript gives the website functionality
  • You can also open Chrome DevTools using keyboard commands. If you’re using windows, you can press Control+Shift+I.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
